# CRUD и агрегатные операции

## Описание проекта
Данный проект представляет собой мини-приложение на Django для управления каталогом товаров.  
Реализованы CRUD-операции для товаров и категорий, фильтрация, сортировка, а также аналитика с использованием агрегатных функций Django ORM.

## Реализованный функционал

### CRUD для товаров
- создание товара
- просмотр списка товаров
- просмотр подробной информации о товаре
- редактирование товара
- удаление товара с подтверждением

### CRUD для категорий
- создание категории
- просмотр списка категорий
- редактирование категории
- удаление категории

### Работа с товарами
Для товара реализованы поля:
- `name` — название товара
- `price` — цена
- `description` — описание
- `created_at` — дата добавления
- `updated_at` — дата обновления
- `category` — категория товара

### Валидация
- поле `price` является обязательным
- реализована проверка, что цена не может быть меньше нуля

### Фильтрация и сортировка
- фильтрация товаров по категории
- фильтрация по диапазону цен
- сортировка по цене по возрастанию и убыванию
- сортировка по дате добавления

### Аналитика
С использованием Django ORM реализованы:
- общее количество товаров
- общая стоимость всех товаров
- средняя цена товаров
- минимальная и максимальная цена
- количество товаров в каждой категории
- общая стоимость товаров по категориям
- средняя, минимальная и максимальная цена по категориям

## Использованные технологии
- Python
- Django
- Django ORM
- SQLite
- Bootstrap 5
